act.qc.qcfilter.QCFilter.set_test¶

QCFilter.set_test(self, var_name, index=None, test_number=None, flag_value=False)[source]¶

Method to set a test/filter in a quality control variable.

Parameters
var_name: str

Data variable name.

index: int or list or numpy array

Index to set test in quality control array. If want to unset all values will need to pass in index of all values.

test_number: int

Test number to remove.

flag_value: boolean

Switch to use flag_values integer quality control.

Examples

index = [0, 1, 2, 30]
ds_object.qcfilter.set_test(
   var_name, index=index, test_number=2)
